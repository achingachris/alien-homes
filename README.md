# HOME APP: ANGULAR(Angular Homes)

Tutorial: https://angular.io/tutorial/first-app

Install Dependencies:

```bash
npm install
```

Install Json Server

```bash
    npm install -g json-server
```

RUN Server:

```bash
json-server --watch db.json
```

RUN Angula App:

```bash
ng serve
```

RUNS ON PORT 4200
